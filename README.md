# News Sentiment Analyzer 📊

A premium Google Chrome Extension that scans news aggregator pages, extracts relevant headlines, and uses Google's **Gemini 2.5 Flash** model to analyze the overall consensus of the articles. 

## 🎥 Video Demonstration
[**▶️ Click Here to Watch the Extension in Action!**](https://youtu.be/1kadp-e64aQ)

## ✨ Features
- **Smart Text Extraction:** Intelligently parses complex DOM structures to cleanly lift out pure article text.
- **Deep Sentiment Analysis:** Automatically identifies the core subject/motion of a page and returns exact metric counts of articles that are **In Favor**, **Against**, or **Neutral**.
- **Detailed Breakdowns:** Generates qualitative summary paragraphs of the arguments for each stance and recommends the single best article to read for that perspective.
- **Modern UI:** Features a sleek aesthetic with animated CSS metrics, clean typography (Inter), and smart layout conditionals.

## 🛠️ Setup Instructions
1. Download this folder.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Toggle on **Developer Mode** in the top right corner.
4. Click **Load unpacked** and select this directory.
5. Click the extension, enter a valid Gemini API Key, and hit "Save"!
6. Navigate to any list of news articles (like Google News or Hacker News) and click **Analyze Current Page**.
